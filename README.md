# Tinymd
This is simple command-line, mardown compiler to get hands-on with rust.

## Command
```
$ cargo run -q test.md
(This will create test.html as output)
```

## Source
This is inspired from a post by [jesselawson.](https://gist.github.com/jesselawson)
